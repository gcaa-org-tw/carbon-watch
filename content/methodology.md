# 企業氣候績效指標制定方法

綠盟的氣候績效指標的核心目標是提供一套可比較、可追蹤、具透明性之評估框架，衡量台灣高排放企業在氣候轉型中的承諾、行動與資訊揭露水準。藉由系統化指標，使公眾、投資人與利害關係人能有效辨識企業在淨零進程中的成熟度、可信度與風險暴露程度。

## 評估架構概覽

評估範圍聚焦於政府盤查名單中之高排放企業，並擴充至具有永續揭露義務或實際排放量具市場影響力者。資料來源以企業公開資訊為主，包括永續報告書、財務報告、官方排放資料、能源使用資訊、董事會與治理文件、情境分析與第三方查驗資料。

評估架構包含三個維度：承諾、行動、揭露。此三維度透過十項績效指標量化。每一指標由多個次級衡量項目構成，以準確反映企業在政策、治理、減碳措施與資料透明度方面的實質進展。

## 三大維度

| 維度            | 含義                        |
| ------------- | ------------------------- |
| 氣候承諾 | 減碳與淨零的長期、中期目標及治理層級的責任劃分   |
| 氣候行動     | 減碳策略、技術與投資、能源結構與供應鏈干預     |
| 資料透明度 | 排放數據透明度、追蹤進度、第三方驗證與資料可比較性 |

## 十項績效指標與定義

| 編號 | 維度 | 指標名稱         | 定義                              |
| -- | ------| ------------ | ------------------------------- |
| 1  | 氣候承諾 | 長期淨零目標       | 訂定至 2050 或更早達到淨零排放之目標並具體化其範疇與邊界 |
| 2  | 氣候承諾 | 中期減碳目標       | 訂定至 2030（或等值基準）之定量減排目標，涵蓋主要排放來源 |
| 3  | 氣候行動 | 範疇一與範疇二排放揭露  | 揭露直接排放與能源間接排放，並提供基準年與年度變化       |
| 4  | 氣候行動 |供應鏈排放管理（範疇三） | 評估、揭露與管理供應鏈及下游使用階段的排放           |
| 5  | 氣候行動 |再生能源與能源轉型    | 設定或執行增加再生能源占比、降低化石燃料依賴之政策或方案    |
| 6  | 氣候行動 |氣候風險治理       | 將氣候風險納入董事會或高階管理層之監督與決策架構        |
| 7  | 氣候行動 |減碳與技術投資      | 對研發、設備汰換、工藝改善、碳捕集或其他低碳技術的投資     |
| 8  | 資料透明度 | 進度追蹤與達成報告    | 定期揭露減碳目標推進程度及未達成時的補充說明與調整策略     |
| 9  | 資料透明度 | 資訊透明度與可驗證性   | 資料完整性、清晰性、可比性與是否由第三方驗證          |
| 10 | 資料透明度 | 利害關係人與供應鏈參與  | 推動供應鏈、員工、顧客或其他利害關係人共同採取氣候行動     |

## 評估流程

1. 資料蒐集
   取得企業永續報告書、盤查登錄資訊、治理文件、能源使用資料、排放清冊、技術投資資料及其他可靠來源。

2. 指標核對
   逐一對照十項指標及其次級衡量項目，檢查企業是否滿足各項標準與細項。

3. 量化與加權
   依據預先定義之量化規則給分並加權整合，產生指標分數、維度分數與整體分數。

4. 多面向分析
   分析指標間的強弱位置、時間序列進步幅度、產業內比較與企業自身年對年變化。

5. 公開呈現
   企業在三維度及十指標下的分數、強項、風險訊號與建議改善領域將於平台展示。

## 方法論適用性與侷限

* 評估完全依賴公開資訊，不對企業未揭露內容進行假設推估。
* 不同行業別之轉型難度、能源結構或技術成熟度差異，目前未納入調整係數。
* 部分指標評估承諾與透明度較具即時性，而實際減碳成效需伴隨多年累積，短期之分數不代表排放絕對下降幅度。
* 政策、技術與揭露準則正快速變化，評估框架將定期更新。

## 方法論的策略邏輯

承諾、行動、揭露三者缺一即產生風險：

* 具承諾但缺乏行動 → 轉型落地存疑
* 具行動但缺乏揭露 → 可信度不足、不可驗證
* 僅揭露而無承諾或行動 → 不足以構成氣候績效

透過三維度與十指標架構，可判斷企業是否具備一致性與整合性，避免單一指標被誤解為整體績效。
